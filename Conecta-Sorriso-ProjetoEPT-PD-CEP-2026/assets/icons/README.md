# Ícones

O protótipo utiliza caracteres e formas simples de interface para evitar dependências adicionais. Esta pasta fica reservada para futuros ícones SVG institucionais.

Antes de adicionar um ícone, confirme se ele possui licença compatível, inclua um texto alternativo quando transmitir informação e use `aria-hidden="true"` quando for apenas decorativo.
